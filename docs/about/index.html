A propos

Ce blog a pour but de présenter Albiziapp, un prototype d’application web pour les inventaires collaboratifs developpé dans le cadre de l’ANR ReVeRIES. Le developpeur principal de l’application est Pierre-Yves Gicquel et elle utilise et souhaite contribuer à l’ecosysteme Open Street Map.

Ce site est repris du package blogdown de R. Le theme provient d’ @jrutheiser/hugo-lithium-theme et modifié par Yihui Xie.
